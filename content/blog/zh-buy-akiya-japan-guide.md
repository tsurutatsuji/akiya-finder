# 外国人如何购买日本空置房（空き家）— 2026年完全指南

日本目前有约900万套空置房屋（日语称为"空き家"），且这一数字仍在持续增长。对于中国买家来说，这是一个前所未有的机遇——以远低于国内房价的价格，购入位于日本的永久产权房产。

本指南将从法律要求、购买流程、费用明细到注意事项，为您全面解析外国人在日本购房的每一个环节。

## 外国人可以在日本买房吗？

**答案是：完全可以。**

与许多国家不同，日本对外国人购买不动产没有任何限制。无论您是否持有日本签证、是否在日本居住，您都可以以个人名义购买日本的土地和建筑物，并获得**永久所有权（所有権）**。

这意味着：

- 不需要日本国籍或永住权
- 不需要在日本设立公司
- 不需要日本的居住地址（但需要委托代理人）
- 土地和建筑都归您所有，可以自由转让和继承

## 购买流程：7个关键步骤

### 第1步：明确购买目的

在开始搜索之前，请先明确您的目的：

- **自住/度假用** — 关注交通便利性、生活设施
- **投资出租** — 关注租金回报率、人口趋势
- **翻新转售** — 关注房屋结构状况、翻新成本

不同目的将直接影响您的选址和预算决策。

### 第2步：搜索房源

日本空置房的信息来源主要有：

- **空き家バンク（空置房数据库）** — 各地方政府运营的官方平台
- **不动产中介网站** — SUUMO、HOME'S、at home等
- **专业海外买家平台** — 如AkiyaFinder，提供多语言支持

### 第3步：实地考察或远程调查

如果无法亲自前往日本，建议：

- 委托当地不动产公司进行现场拍照和视频录制
- 了解房屋的**建筑年份**（1981年以后建造的符合新抗震标准）
- 确认**基础设施状况**（水、电、燃气、下水道）
- 检查是否存在**白蚁损害**或**漏水问题**

### 第4步：提交购买申请

通过不动产中介向卖方提交**购入申込書（购买申请书）**。这一步通常不具有法律约束力，但表示您的购买意向。

### 第5步：签署买卖合同

在日本持有不动产执照的**宅地建物取引士**（注册不动产交易师）将向您详细解释合同条款。合同签署时通常需要支付**定金（手付金）**，一般为房价的5-10%。

**重要提示**：如果您不在日本，可以委托代理人签署。需要提前在中国的日本大使馆或领事馆办理**委任状**的公证。

### 第6步：办理过户登记

在日本，房产过户需要通过**司法书士**（类似中国的公证员）办理不动产登记。登记完成后，您即正式成为房产的所有者。

### 第7步：支付尾款及各项税费

在过户当天或之前，您需要支付全部房款及相关费用。

## 购房费用明细

除了房价本身，您还需要准备以下费用：

| 费用项目 | 大约金额 |
|---------|---------|
| 不动产中介费 | 房价的3% + 6万日元 + 消费税 |
| 登记免许税 | 房价的1.5-2% |
| 不动产取得税 | 固定资产评估额的3-4% |
| 司法书士费用 | 5-15万日元 |
| 印花税 | 1-6万日元 |
| 固定资产税（年度） | 固定资产评估额的1.4% |

**一般来说，购房总附加费用约为房价的6-10%。**

## 汇款注意事项

从中国向日本汇款购房，需要注意以下几点：

- 中国个人年度购汇额度为5万美元，大额购房可能需要多次汇款
- 建议通过银行电汇（TT）至卖方或司法书士的指定账户
- 提前确认日本银行是否接受海外汇款
- 保留所有汇款凭证，用于日后的税务申报

## 购买空置房的特别注意事项

空置房与普通二手房不同，需要额外注意：

1. **建筑年份** — 1981年以前的房屋不符合现行抗震标准，翻新成本可能很高
2. **长期空置的隐患** — 水管锈蚀、屋顶漏水、白蚁侵蚀、杂草丛生
3. **接道义务** — 日本法律要求建筑用地必须面向4米以上的道路至少2米宽，不满足条件的土地无法重建
4. **都市计划区域** — 部分区域禁止或限制新建建筑
5. **固定资产税减免可能失效** — 长期空置且荒废的房屋可能被认定为"特定空家"，税额增加至6倍

## 是否需要签证？

购买房产本身**不需要签证**，也**不会自动获得日本的居住签证**。日本目前没有"购房移民"政策。

但如果您希望长期居住，可以考虑：

- **经营管理签证** — 在日本开设公司（如民宿经营），投资500万日元以上
- **高度人才签证** — 根据积分制度申请

日本空置房对中国买家来说是一个值得认真考虑的投资机会。永久产权、无购买限制、价格低廉——这些优势在全球房产市场中极为罕见。关键在于做好充分的调研，找到可靠的当地合作伙伴。
