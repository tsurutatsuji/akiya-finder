## 調査2: 紹介料を得やすい条件

### 1. 紹介料の法的条件（宅建免許なしで合法）

**2016年 経済産業省が合法と認定:**

| 項目 | 詳細 |
|------|------|
| **合法な行為** | 「業者（宅建業者）を紹介する」こと。顧客を不動産会社に繋ぐだけならOK |
| **違法な行為** | 「物件を紹介する」こと。物件の説明・条件交渉・契約調整に関与すると宅建業法違反 |
| **反復継続** | 合法。業者への顧客紹介を反復継続しても宅建業法に抵触しないと確認済み |
| **契約の明確化** | 業務提携契約の内容を詳細かつ明確に取り決める必要あり |
| **税務** | 紹介料合計20万円超で雑所得として確定申告必要 |

**AkiyaFinderがやるべきこと:**
- 外国人バイヤーを集客し、**宅建業者を紹介**する（物件紹介ではない）
- 「この業者に相談してください」と繋ぐだけ。物件の説明・交渉はしない
- 業務提携契約書を業者と事前に締結する

**AkiyaFinderがやってはいけないこと:**
- 物件の詳細説明（間取り・価格交渉・条件提示）
- 取引条件の交渉・調整
- 契約書の作成・関与
- 初回面談以降の同席（初回のみOK）

### 2. 紹介料の相場

| 取引タイプ | 紹介料相場 | AkiyaFinder想定 |
|-----------|-----------|----------------|
| **売買（一般）** | 物件価格の0.5%〜2% | 500万円の物件 → 2.5万〜10万円 |
| **売買（大手）** | 上限5万円（現金or商品券） | 大手は渋い |
| **買取** | 買取価格の1%〜3% | 300万円の買取 → 3万〜9万円 |
| **賃貸** | 月額賃料の10%〜50% | 賃料10万円 → 1万〜5万円 |

**AkiyaFinderの場合の試算:**
- 平均物件価格500万円 × 紹介料1% = **5万円/件**
- 月10件成約 = **月50万円**
- 月20件成約 = **月100万円**

### 3. 紹介を受け入れやすい業者の条件

| 条件 | 理由 |
|------|------|
| **外国人対応に慣れていない業者** | 自分では外国人集客できないので、紹介してくれるなら喜ぶ |
| **地方の小規模業者** | 大手と違い集客チャネルが限られる。紹介は貴重 |
| **空き家・格安物件専門** | 利益率が低いため、1件でも多く成約したい |
| **英語ができないが外国人の問い合わせがある業者** | 言語の壁で機会損失している。通訳兼紹介なら価値が高い |
| **観光地・リゾート地の業者** | 外国人需要があるが対応しきれていない |

### 4. 紹介料交渉で有利になる条件

| 条件 | 効果 |
|------|------|
| **継続的に紹介できる実績** | 「月X件紹介します」と約束できると交渉力UP |
| **質の高いリード** | 冷やかしでなく、購入意欲・予算が明確なバイヤーを紹介 |
| **言語サポート付き** | 英語/中国語の通訳・翻訳を提供すると付加価値 |
| **リモート対応** | 海外在住バイヤーの初期相談・物件選定をサポート |
| **成約率の高さ** | 紹介→成約率が高ければ、紹介料率の引き上げ交渉が可能 |

### 5. 個人参入の制約・限界

| 制約 | 対策 |
|------|------|
| **物件説明NG** | あくまで「業者紹介」に徹する。サイト上でも「この業者に相談」導線にする |
| **安定収入が難しい** | 複数業者と提携し、地域を分散させる |
| **業者との信頼構築** | 最初は無料で紹介し実績を作る。成約後に紹介料交渉 |
| **確定申告** | 20万円超で雑所得。事業として本格化するなら個人事業主登録 |

## 戦略的インサイト

### AkiyaFinderが狙うべきスイートスポット

1. **地域**: 長野（白馬以外の松本・上田）、京都（町家）、広島（尾道・瀬戸内）、北海道（札幌周辺・ニセコ以外）、静岡（伊豆・熱海）
2. **価格帯**: 50〜500万円の物件（衝動買い可能な価格帯）
3. **物件タイプ**: 古民家・町家が最も需要が高い
4. **ターゲット**: 英語圏の30〜60代。セカンドハウス＋民泊目的
5. **提携先**: Aランクの業者（Arrows、Heritage Homes、ResCom）から優先的にアプローチ

### 中国人バイヤーへの対応

- 空き家（800万円以下）は中国人富裕層のメインターゲットではない
- ただし、富士山ビュー・温泉付きの500〜1000万円帯なら需要あり
- 中国語対応はHachise（京都）が最有力パートナー

### 紹介モデルの最適化

- **初期**: 無料で3〜5件紹介して実績を作る → 成約後に紹介料交渉
- **中期**: 月10件以上の安定紹介で紹介料率1〜2%を確保
- **長期**: 「AkiyaFinder紹介」ブランドで業者側から提携依頼が来る状態を目指す
